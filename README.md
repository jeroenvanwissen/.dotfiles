# Dotfiles

My personal dotfiles for macOS development environment.

## What's Inside

- **Fish Shell**: Custom shell configuration with Fisher plugin manager
- **Helix**: Modern text editor with comprehensive language server support
- **Tmux**: Terminal multiplexer with catppuccin theme
- **Starship**: Cross-shell prompt configuration
- **Kitty**: GPU-based terminal emulator settings (macOS only)

## Installation

### macOS

Clone the repository and run the macOS installer:

```bash
git clone https://github.com/jeroenvanwissen/.dotfiles.git ~/.dotfiles
cd ~/.dotfiles
./install_macos.sh
```

### Ubuntu/WSL

Clone the repository and run the Ubuntu installer:

```bash  
git clone https://github.com/jeroenvanwissen/.dotfiles.git ~/.dotfiles
cd ~/.dotfiles
./install_ubuntu_wsl.sh
```

This will set up all the necessary tools, symlinks and configurations.
